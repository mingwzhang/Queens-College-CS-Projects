
HOW TO RUN:

SERVER (1ST):

	1. Open Terminal / Command Prompt / Powershell			

	2. Enter -> cd "[directory to Zhang_server folder]"		// Change directory to Zhang_server folder 

	3. Enter -> javac *.java					// Compiles all .java files in Zhang_server folder 
	
	4. Enter -> java _MainServer					// Executes 


CLIENT (2ND):

	1. Open Terminal / Command Prompt / Powershell			

	2. Enter -> cd "[directory to Zhang_client folder]"		// Change directory to Zhang_client folder 

	3. Enter -> javac *.java					// Compiles all .java files in Zhang_client folder 
	
	4. Enter -> java _ClientLauncher				// Executes
